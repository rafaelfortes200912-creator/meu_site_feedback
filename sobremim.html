Rafael | Design & Code

Do Huambo para o mundo digital, sou um desenvolvedor que não abre mão da liberdade criativa. Especialista em dar vida a interfaces modernas, meu trabalho é pautado pela filosofia de que o melhor software é aquele feito sob medida. Como entusiasta de sistemas abertos e design minimalista, encaro cada desafio como uma oportunidade de aplicar meu perfeccionismo técnico. Se o objetivo é criar algo que se destaque pela elegância e pela performance, eu garanto que cada linha de HTML, CSS e JavaScript terá o meu selo de qualidade.
